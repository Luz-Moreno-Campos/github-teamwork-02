## 
# GitHub Testing 2
A collaborative web development project designed to practice Git workflows, branching strategies, and semantic HTML/CSS structure.
## Project Overview
This Project serves as a practice environment for a team of developer to coordinate on a single codebase. It focuses on building a clean, accessible informational page about web standards and version control.
## Built With
* **HTML** - Semantic structure (using tags like `<main>`, `<section>`, and `<blockquote>`)
* **CSS** - Custom styling and layout
* **Font Awesome** - Professional iconography
* **Git/GitHub** - Version control and team collaboration
